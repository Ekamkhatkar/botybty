## ğŸ“‹ Command Documentation

### Review System

The review system allows server members to leave reviews for each other with star ratings and comments.

- **/review setup** - Set up the review system by specifying a channel where reviews will be displayed.
- **/review add** - Add a review for a specific user with a star rating (1-5) and a message.
- **/review view** - View all reviews for a specific user, including their average rating.
- **/review restore** - Restore all reviews to the designated channel if they've been purged or deleted.
